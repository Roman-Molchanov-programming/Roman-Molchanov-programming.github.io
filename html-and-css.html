CSS - Cascading Style Sheets

'.', '#' и др. это - селекторы (selectors). (от анл. select - выбрать).

Стандартные ситуации выбора

  1. Body - единственный уникальный элемент в html. Стилизация body работает на всю страницу.
    body {}
  2. бывает необходимость выделить все параграфы
    p {}
  3. Необходимо выделить несколько параграфов одного типа
    задать тип параграфа при помощи аттрибута "class", напрример class=comment p.comment {}
  4. Выделить один уникальный параграф
    задать уникальное имя при помощи id, например id=someunique
    #someunique {}
  5. Выделить любой элемент, относящцйся к определенному типу (класс) h1 class = class1
    p class = class1
    .class1 {}
1. Сделать background страницы - серым. 2. Сделать все параграфы (в отличие от ul, ol, h1 и др.) - синим. 3. Сделать все парагра - коментарии - зеленым. 4. Слелать параграф для слайдера - красным.
