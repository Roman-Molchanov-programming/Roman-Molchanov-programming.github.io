Branch-Commands

  • Создание бранча - git checkout -b название бранча (M-17-correct-attribute-order-for-meta-tag)
  • Названия доступных бранчей - git branch
  • Проверка всех бранчей - git branch -r
  • Проверка в изменениях в файлах - git diff (q - для выхода)
  • Выбор бранча - git checkout название бранча
  • Пуш бранча - git push название
